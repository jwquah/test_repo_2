### Lambda
Lambda can be called as "one line function". Essentially you run it once, in a single line.

#### Example(s)
```python
add = lambda x,y : x + y
print ('Addition result      : {}'.format(add(3,4)))
### Addition result      : 7
mult = lambda x,y : x * y
print('Multiplication result: {}'.format(mult(3,4)))
### Multiplication result: 12
```
