## Math Operation


```python
a = 2 * 3
print("Value: {}, and Type: {}".format(a, type(a)))
```

    Value: 6, and Type: <class 'int'>



```python
a = 2 ** 3
print("Value: {}, and Type: {}".format(a, type(a)))
```

    Value: 8, and Type: <class 'int'>



```python
a = 3 / 2.0
print("Value: {}, and Type: {}".format(a, type(a)))
```

    Value: 1.5, and Type: <class 'float'>
